# O que é HTML?
- HTML é composto de TAGS

~~~~html
<!-- Pirmeira tag a ser colocada, indica ao navegador que está utilizando html5 -->
<!DOCTYPE html>
<!-- tag html: início do código html -->
<html>
    <head>
        <!-- no head vai tags de configuração (títulos e meta tags) e tags que chamam arquivos -->
         <title>Primeiro HTML</title>
    </head>
    <body>
        <h1>Hello Word!</h1>
    </body>
</html>
~~~~
